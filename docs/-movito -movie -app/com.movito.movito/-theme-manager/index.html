ThemeManager

object ThemeManager

Singleton manager for application theme state and persistence.

This object serves as the single source of truth for theme state throughout the app, providing:

  • Reactive theme state via StateFlow for real-time UI updates

  • Theme preference persistence using SharedPreferences

  • App-wide theme consistency across all activities and fragments

Theme states:

  • false: Light theme

  • true: Dark theme

Architecture rationale: This singleton pattern is appropriate because theme state is truly global - all parts of the app need consistent access to the current theme. ViewModels wrap this singleton to provide lifecycle-aware observation.
