TmdbApi

interface TmdbApi

Retrofit interface defining all TMDB API endpoints used by the Movito app.

This interface declares suspend functions for asynchronous API calls using Retrofit. All methods are coroutine-friendly and return deserialized response objects.

Key API categories:

  • Movie discovery and filtering

  • Genre information

  • Movie search

  • Video content (trailers)

  • Movie recommendations

Author: Movito Development Team Member Mohamed Ibrahim
